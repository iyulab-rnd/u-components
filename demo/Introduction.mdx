# U-COMPONENTS: 웹 컴포넌트 라이브러리

iyulab에서 사용하는 웹 컴포넌트 라이브러리입니다.

## 설치하기

```bash
npm install @iyulab/u-components
```

## 모두 사용하기

```typescript
import "@iyulab/u-components";
```

## 특정 컴포넌트 사용하기

```typescript
import "@iyulab/u-components/components/buttons";
```

## 컴포넌트 목록

- [Buttons](./buttons.md)
- [Inputs](./inputs.md)
- [Modals](./modals.md)
